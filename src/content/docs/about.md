## Что такое Munidoc?

**Munidoc** — это открытая база знаний и набор инструментов для специалистов в сфере муниципального права. 

### Наши цели
1. Систематизация разрозненных знаний о работе ОМСУ (Органов местного самоуправления).
2. Создание единого стандарта качества для правовых актов.
3. Внедрение современных технологий (AI, автоматизация) в рутинную работу юриста.

### Для кого этот проект
* Для юристов администраций и советов депутатов.
* Для муниципальных служащих.
* Для депутатов представительных органов.
